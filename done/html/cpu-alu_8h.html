some useful macros:

  • combine_flags_set_A: used in do_cpu_arithm to combine ALU flags, set F and set A
  • extract_carry: used in do_cpu_arithm to to extract carry bit from opcode
  • do_cpu_arithm: does(=applies) operation OP from ALU on CPU using ARG as operation argument and FLAGS_SRC flag sources (see for instance ADD_FLAGS_SRC macro above: this are the flag sources to be used for ADD operations.)
